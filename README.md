# AI_APP

Dự án gồm hai phần chính:

- `fontend/`: ứng dụng web Next.js.
- `backend/`: thư mục để đặt dịch vụ phía máy chủ (hiện chưa cấu hình).

## Khởi chạy nhanh

1. Cài đặt phụ thuộc cho phần giao diện:

```bash
cd fontend
npm install
npm run dev
```

2. Thêm các tệp cấu hình bí mật (ví dụ `.env`) vào đúng thư mục nhưng không đưa lên kho mã.

## Ghi chú

- Thêm tài liệu chi tiết hơn tại đây khi dự án phát triển (cách chạy backend, dữ liệu môi trường, v.v.).
- Đảm bảo không commit thông tin nhạy cảm (token, mật khẩu, khóa API).
